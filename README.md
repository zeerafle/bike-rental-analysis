# Bike Rental Dashboard

## Setup Environment

### With uv (recommended)

Install [uv](https://docs.astral.sh/uv/) jika belum

```bash
uv sync
uv run streamlit run dashboard.py
```

### With pip

```bash
python -m venv .venv
.\.venv\Scripts\activate
# linux: ./.venv/bin/activate
pip install -r requirements.txt
streamlit run dashboard.py
```
